Remove Customizer

Follow below steps to remove that button

When using Html
1. Every html file has a customizer section.You just remove that section 
<li class="cursor-pointer flex customizer-btn items-center justify-center   px-[calc(7px_+_(14_-_7)_*_((100vw_-_320px)_/_(1920_-_320)))] rtl:border-l sm:rtl:last:pl-[calc(7px_+_(14_-_7)_*_((100vw_-_320px)_/_(1920_-_320)))] 
  rtl:border-r-0 first:pl-0 last:pr-0 last:border-none sm:last:pr-[calc(7px_+_(14_-_7)_*_((100vw_-_320px)_/_(1920_-_320)))] 
  sm:last:border-r sm:last:border-solid border-r border-border-light dark:border-border-dark"> ...</li>
2. There is a script in the same file, remove the script from too 
<!-- Template Customizer --><script  src="../../assets/js/customizer.js"></script>
When using pug

Add the code in the first line of the pug file to remove it from the pug file

"-var customizer = false;"
